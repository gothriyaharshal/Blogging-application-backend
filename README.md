# Blogging Application

This project is a backend blogging application developed using Java and Spring Boot.

## Technologies Used
- Java
- Spring Boot
- MySQL
- JPA / Hibernate

## Features
- Create blog posts
- Read blog posts
- Update blog posts
- Delete blog posts

## Description
This project demonstrates basic backend development concepts such as REST APIs, database integration, and CRUD operations using Spring Boot.
